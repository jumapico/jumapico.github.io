Complementos Firefox

Habilitados:

  • Dark Reader Permite habilitar el Dark mode por sitio.
  • Firefox Multi-Account Containers Separa cookies por contenedor.
  • Greasemonkey User script manager.
  • HTTPS Everywhere Intentar utilizar HTTPS automaticamente al navegar.
  • NoScript Habilitar Javascript solo en los sitios deseados.
  • Privacy Badger Bloquear trackers.
  • SingleFile Descargar páginas web completas en un solo archivo.
  • To Google Translate Traducir porciones de texto de una página.
  • Wayback Machine Obtener una versión archivada del sitio.

Deshabilitados:

  • Cookie Quick Manager Manejo de cookies.
  • RESTED Helper para hacer pedidos REST (ver httpie para alternativa en consola).
  • User-Agent Switcher and Manager Modificar el User-Agent.

Scripts greasemonkey (userscripts)

  • Google: Direct Links for Pages and Images Show direct links to web pages and images for google result.
  • fix-roundcube-compose-overlap: Fix overlapping issue when compose email in roundcube webmail.
  • Medium Sticky Shit: Remove fixed footer from medium site.
  • Hide Hangouts background: Hide Hangouts background.
