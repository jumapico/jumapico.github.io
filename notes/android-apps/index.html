Instalación de F-Droid

Si el dispositivo trae instalado un navegador por defecto utilizarlo para descargar el apk. Luego dar permiso al navegador para que instale aplicaciones de origen desconocido e instalar el apk. Una vez instalado F-Droid quitar permiso para instalar aplicaciones de origen desconocido.

Si el dispositivo no trae instalado un navegador por defecto se deberá instalar la aplicación F-Droid desde el PC.
