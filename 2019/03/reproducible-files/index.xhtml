Archivos reproducibles

Fecha de agregado: 2019-03-01

Palabras clave: reproducible builds

Este tip es creado para tener un lugar fácil en el cual buscar cuando no se recuerdan exactamente los comandos a utilizar.

Comandos utilizados frecuentemente para la creación de archivos reproducibles. En su mayoría obtenidos de 1 y 2.

Creación de archivo tar (ver 3)

  • Entradas ordenadas

  1. Versión 1 - tar no posee la opción --sort

    $ find src -print0 | LC_ALL=C sort -z | \
    tar --no-recursion --null -T - -cf archive.tar

  2. Versión 2 - tar posee la opción --sort

    $ tar --sort=name -cf archive.tar src
