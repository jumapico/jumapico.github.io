Resetear extensiones de gnome

Fecha de agregado: 2019-01-02

Palabras clave: gnome mate

Hace días que tengo problemas con las extensiones de gnome en varias de mis PC's usando gnome classic:

  • PC 1 (gnome 3.22): no aparece la windows list
  • PC 2 (gnome 3.30): solo aparece el fondo de pantalla. No aparece la topbar ni la windows list y cuando se abre una ventana esta aparece sin decoraciones.

La solución que encontré para corregir los problemas anteriores fué resetear las extensiones de gnome shell, eliminando el directorio ~/.local/share/gnome-shell:

$ rm -fr ~/.local/share/gnome-shell

Obviamente con lo anterior se pierden las extensiones que pudieron haber sido instaladas por el usuario.

Observaciones

Realmente asombrado con el hecho de que de un día para otro, sin haber realizado cambios en la configuración del escritorio (o de las extensiones), gnome deje de funcionar como lo venía haciendo.

A raíz de los problemas anteriores es que comencé a utilizar mate como escritorio, que si bien no es tan atractivo visualmente, al carecer de extensiones estoy seguro que no me va volver a suceder que entre un día a mi sesión y esté imposibilitado de utilizar el escritorio, o me falte un elemento importante como la lista de ventanas. Para quien no lo sepa, mate es un fork de gnome 2.

Último comentario sobre gnome: no me agrada pensar que javascript es el único lenguaje para programar las extensiones. Realmente hubiera preferido tener lua como opción.
