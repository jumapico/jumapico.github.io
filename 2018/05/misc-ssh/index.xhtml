Varios ssh

Fecha de agregado: 2018-05-02

Palabras clave: ssh

Varios ssh:

  • Cambiar password del keyfile:

    $ ssh-keygen -p -f keyfile

  • Regenerar el archivo public key a partir del keyfile:

    $ ssh-keygen -y -f input_keyfile

    Nota: el key pair mencionado en la documentación de amazon ec2 es en realidad el keyfile generado con ssh-keygen con extension pem. El public key se genera a partir de este, por lo que no es necesario tenerlo almacenado.
