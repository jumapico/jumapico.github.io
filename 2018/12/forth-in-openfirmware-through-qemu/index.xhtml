Forth en Open Firmware mediante qemu

Fecha de agregado: 2018-12-31

Palabras clave: forth openfirmware qemu

Instalar en debian el paquete qemu-system-ppc:

$ sudo apt-get install -V qemu-system-ppc

Para utilizar Forth (en Open Firmware) ejecutar:

$ qemu-system-ppc64 \
    --net none \
    -nographic \
    -vga none \
    -chardev stdio,mux=on,id=char0 \
    -mon chardev=char0,mode=readline \
    -serial chardev:char0

El comando anterior ejecuta QEMU en la terminal, redirigiendo la entrada/salida a la consola, multiplexandola entre el monitor de qemu y el puerto virtual serial.

Para alternar entre el monitor de qemu y el puerto virtual serial utilizar Ctrl-a c. Para mostrar la ayuda utilizar Ctrl-a h.

Nota

La idea de utilizar el interprete de Forth en Open Firmware mediante QEMU surgió a partir de la imagen del día 04, Day 4 - Fun with Forth, del QEMU Advent Calendar 2018.
