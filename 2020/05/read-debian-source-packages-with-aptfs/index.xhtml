Leyendo paquetes fuente de debian con aptfs

Fecha de agregado: 2020-05-18

Palabras clave: debian aptfs

Para poder utilizar aptfs en debian es necesario tener agregado por lo menos un repositorio de paquetes fuente, sinó luego de montar aptfs el directorio se verá vacío.

Para instalar aptfs y configurar los fuentes de debian testing (suponiendo que no están ya agregados a apt):

$ sudo apt-get install -Vy aptfs
$ echo 'deb-src http://deb.debian.org/debian testing main contrib non-free' \
   | sudo tee /etc/apt/sources.list.d/testing-src.list > /dev/null
$ sudo apt-get update

Para montar aptfs (usando un máximo de 10 paquetes fuentes desempaquetados a la vez):

$ mkdir -p ~/workspace-debian-sources
$ mount.aptfs -o max_unpacked_packages=10 ~/workspace-debian-sources
