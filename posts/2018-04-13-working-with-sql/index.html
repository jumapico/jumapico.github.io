Trabajar con sql

2018-04-13

Como escribir sql:

  • Con PostgreSQL:

    $ psql --variable "<name>=<value>" -f query.sql database

    o en una sesión psql:

    > \cd path/to/my/sources
> \set name value
> \i query.sql

  • Con Oracle

    En una sesión sqlcl usando bind variables:

    SQL> cd path/to/my/sources
SQL> var name datatype
SQL> exec :name := value
SQL> @query.sql
