Servidor HTTP standalone

2018/07/04

Se pueden servir los archivos de un directorio utilizando la clase SimpleHTTPRequestHandler de la librería http.server de python3 desde la linea de comandos:

$ python3 -m http.server

Otros

  • busybox:

    $ busybox httpd -f -p 8000

  • php:

    $ php -S localhost:8000

  • ruby (lista directorios):

    $ ruby -run -ehttpd . -p8000
