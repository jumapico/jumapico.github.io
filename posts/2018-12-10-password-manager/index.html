Manejador de contraseñas

2018-12-10

Para manejar las contraseñas se puede utilizar keepassxc, recomendado por la Electronic Frontier Foundation y disponible en Debian:

$ sudo apt-get install -V keepassxc

Como alternativa se puede ver el programa pass - the standard unix password manager, también disponible en Debian y que sigue más de cerca la filosofía unix, utilizando varios programas ya disponibles:

$ sudo apt-get install -V pass
