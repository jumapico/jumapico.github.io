Reinstalando windows 10

2021-08-04

Contexto

Estos días tuve un familiar con un problema en la laptop. Especificamente un fallo en el disco duro, por lo que debí reinstalarle windows 10.

Ya que es posible que el disco siga con problemas y a futuro deba volver a realizar el procedimiento (ya que por ahora no piensa cambiarlo), agrego las indicaciones seguidas para que la próxima vez sea más sencillo de realizar.

Sobre la licencia de windows

En varios modelos de laptops, la licencia de windows viene embebida en el BIOS y alcanza con instalar la última versión de windows 10. Durante la instalación (o una vez instalado, no investigué lo suficiente), la licencia se lee del BIOS.

Para verificar que la laptop cuente con dicha licencia se puede utilizar una imagen booteable de linux y revisar el archivo /sys/firmware/acpi/tables/MSDM.

Por más detalles ver https://www.cyberciti.biz/faq/linux-find-windows-10-oem-product-key-command/.

Preparar el pendrive de instalación

  1. Descargar el ISO de windows 10. Para ello ir al sitio https://www.microsoft.com/en-us/software-download/windows10ISO

  2. Utilizar el programa bootiso para crear un pendrive booteable:

    $ curl -L -O https://raw.githubusercontent.com/jsamr/bootiso/v4.2.0/bootiso
$ chmod +x bootiso
$ ./bootiso -p ~/Downloads/Win10_21H1_Spanish_x64.iso  # inspeccionar si el
                                                       # iso es booteable y
                                                       # listar usb's
$ ./bootiso ~/Downloads/Win10_21H1_Spanish_x64.iso  # crear el pendrive

Configuracion de windows

  1. Deshabilitar las actualizaciones automáticas utilizando el programa services.msc. Esto se realiza a pedido del usuario. Ver https://www.alphr.com/microsoft/microsoft-windows-10/1003724/how-to-stop-the-windows-10-upgrade-from-downloading/.
