Medir velocidad de red

2018/06/01

Medir velocidad (aproximada) de red entre PC’s utilizando netcat y nload

PC servidor:

$ nc -l port > /tmp/test.out

PC cliente, consola 1:

$ sudo nload interface

PC cliente, consola 2:

$ nc server-ip port < /path/to/file

NOTA: Tener en cuenta que según las políticas de red algunos puertos pueden estar filtrados. Si ese es el caso y es posible utilizar el puerto 80 este puede ser una opción.
