Recursos para aprender perl

2019-10-07

Los otros días le pedí a mi amigo Jean Pierre (programador senior, si no más, en perl entre otras cosas) que me diera algunas recomendaciones ya que estoy aprendiendo perl y cuento con él para que me facilite el camino de aprendizaje.

Programas

  • perlbrew (App::perlbrew), admin-free perl installation management tool.
  • perltidy (Perl::Tidy), a perl script indenter and reformatter.
  • cpanm (App::cpanminus), get, unpack, build and install modules from CPAN.
  • countperl (Perl::Metrics::Simple), Count packages, subs, lines, etc. of many files.
  • perlcritic (Perl::Critic), Critique Perl source code for best-practices.
  • Try::Tiny, Minimal try/catch with proper preservation of $@.

Para instalar los anteriores en debian hacer:

$ sudo apt-get install -Vy \
    perlbrew \
    perltidy \
    cpanminus \
    libperl-metrics-simple-perl \
    libperl-critic-perl \
    libtry-tiny-perl

Libros

Si bien estoy utilizando como referencia el libro Learning Perl, los siguientes también me fueron recomendados:

  • Perl Testing - A Developer's Notebook
  • Perl Best Practices
