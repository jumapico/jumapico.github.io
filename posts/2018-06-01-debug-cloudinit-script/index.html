Depurar script de cloudinit

Borrar los archivos:

/var/lib/cloud/instances/$UUID/boot-finished
/var/lib/cloud/instances/$UUID/sem/config_scripts_user

Luego ejecutar el módulo final para ejecutar el script user_data:

cloud-init modules --mode final

NOTA: Los cambios se mantienen, por lo que se debe escribir código idempotente.
