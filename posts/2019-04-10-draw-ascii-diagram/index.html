Creación de diagramas ascii

2019/04/10

Asciio es una aplicación perl que permite crear diagramas en ASCII.

Para instalarla en Debian ejecutar:

$ sudo apt-get update
$ sudo apt-get install -Vy asciio

Un screenshot se puede encontrar el sitio debian screenshots.
