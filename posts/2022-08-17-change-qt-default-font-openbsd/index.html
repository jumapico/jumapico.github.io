Cambiar fuente por defecto de QT en OpenBSD

2022-08-17

Estoy utilizando mate-desktop en OpenBSD el cual está basado en GTK.

Para poder cambiar las fuentes utilizadas por programas que utilicen la librería Qt:

  1. Instalar el paquete qt5ct:

    $ doas pkg_add qt5ct

  2. Agregar la variable de entorno QT_QPA_PLATFORMTHEME=qt5ct al comienzo de ~/.xsession como indica la descripción del paquete:

    $ pkg_info -d qt5ct
...
Set QT_QPA_PLATFORMTHEME=qt5ct in the environment to use it.
...

  3. Configurar las fuentes ejecutando qt5ct (pestaña Fonts).
